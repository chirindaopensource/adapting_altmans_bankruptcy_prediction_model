# adapting_altmans_bankruptcy_prediction_model
End-to-End Python implementation of a bankruptcy prediction method which adapts Altman's Z-Score to Compositional Data Analysis (Keivani et al., 2026). Benefits: Uses the Aitchison simplex to eliminate outliers &amp; asymmetry in financial ratios. Pipeline: log-ratio EM imputation, pairwise log-ratios, ML classifiers (Logit/k-NN/RF), validation.
